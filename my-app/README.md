## DESCRIPTION
一个基于 `Node` + `React` + `Webpack` + `MongoDB` 实现的 ToDoList 应用。


## FUNCTION

- Add ToDo
- Delete ToDo

## INSTALL
```js
npm install
```

## START
```js
mongod --dbpath D:/mongodb/data
```

```js
webpack -w
```

```js
npm start
```
